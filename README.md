# API-lab-7-Zaitseva-Valeria-
# Учебный проект: API и Tkinter

Проект состоит из трех частей:

1. Погода (`weather.py`)  
   - Использует OpenWeatherMap API
   - Выводит текущую погоду в любом городе

2. Праздники и словарь (`holidays_dict.py`)  
   - Использует Holiday API и Dictionary API  
   - Выводит праздники по стране и информацию о словах, но только на английском

3. Генератор картинок (`image_generator.py`)  
   - Использует Tkinter и API https://nekos.best  
   - Показывает аниме-картинку и меняет её по кнопке
